Веб-сайт для мобильных устройств, для распространения и удобного использования специальных нот для гитары (табов)

Целевая аудитория - музыканты на классической и акустической гитарах в инструментальных жанрах

Запланированный функционал первых версий:
1. Возможность публикации нот пльзователями (в дальнейшем с системой авторизации и модерацией)
2. Система рейтинга произведений
3. Возможность синхронного воспроизведения оригинального звука с нотами
4. Собственно доступ ко всем опубликованным нотам, с системой поиска

Перспективы развития:
1. Создание специализированного звукого движка для классической гитары, для синтеза звуковой дорожки по нотам
2. Создание мобильного приложения 
3. Коллаборации с классическими гитаристами для поиска аудитории 
4. Возможность добавления функции платного доступа авторам нот (крайне большое количество музыкантов использует этот способ монетизации и это поможет привлечь их на платформу, возможно с введением партнерской программы)

Система монетизации:
Краудфандинг. Доступ ко всему функционалу полностью бесплатный, что обоснованно анализом конкурентов и взглядами автора
